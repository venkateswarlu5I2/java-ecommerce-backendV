# 🛒 Java Ecommerce Backend

A scalable and secure **Ecommerce Backend API** built using **Java and Spring Boot**.
This project provides RESTful APIs to manage users, products, carts, and orders for an online shopping platform.

---

## 🚀 Features

* 🔐 User Authentication & Authorization
* 🛍️ Product Management
* 🛒 Shopping Cart System
* 📦 Order Processing
* 📊 Admin Management
* 🔎 Product Search & Filtering
* 📡 RESTful API Architecture

---

## 🧰 Tech Stack

* **Java**
* **Spring Boot**
* **Spring Data JPA**
* **MySQL**
* **Maven**
* **REST API**
* **Git & GitHub**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/java-ecommerce-backend.git
```

### 2️⃣ Navigate to project folder

```
cd java-ecommerce-backend
```

### 3️⃣ Run the project

```
mvn spring-boot:run
```
